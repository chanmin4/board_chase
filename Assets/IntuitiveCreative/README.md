## Usage

All Intuitive Creative tools follow the same general workflow and are designed to be used directly from the Unity Editor.

### Opening a Tool
1. Select one or more supported assets in the **Project** window.
2. Right-click the selection.
3. Navigate to **Intuitive Creative** and choose the desired tool.

Tools are also available from the top menu under  
**Tools → Intuitive Creative**.

---

### Editing Assets
Once a tool is open:

- Adjust values using the provided **sliders and controls**.
- Batch tools apply the same settings to all selected assets.
- Most tools provide a live or near-live preview to help evaluate changes before saving.

---

### Saving Options
Before applying changes, choose how the modified assets should be saved:

- **Overwrite Original**  
  Replaces the original asset with the modified version.

- **Backup and Overwrite**  
  Creates a backup copy of the original asset before overwriting it.

- **Save as Copy**  
  Saves the modified asset as a new file and leaves the original unchanged.

After saving, assets are automatically reimported by Unity.

---

### Batch Tools
Batch tools operate on multiple selected assets at once:

1. Select multiple compatible assets in the Project window.
2. Open the corresponding batch tool from the **Intuitive Creative** menu.
3. Configure the desired settings.
4. Apply the changes to process all selected assets.

Batch tools use the same saving options as single-asset tools.

---

### Notes
- These tools are **Editor-only** and do not add runtime dependencies.
- Some assets may need to be marked as **readable** to be processed.
- For best results, work on source assets rather than heavily compressed files.
