# Quick Start: Intuitive Audio Adjust

## Demo Assets
Demo files for this tool:
- `Assets/IntuitiveCreative/Demos/IntuitiveAudioAdjust/inputs_raw`
- `Assets/IntuitiveCreative/Demos/IntuitiveAudioAdjust/outputs_processed`

## Try It
1. In the Project window, select any WAV file in `inputs_raw`.
2. Right-click the selection and choose `Intuitive Creative → Audio Adjust`.
3. Apply trim, fade, or volume changes.
4. Compare your result with the matching file in `outputs_processed`.
